# AJAX

AJAX (Asynchronous JavaScript and XML) practice :
* Working with APIs
* Fetch API
* Working with Axios

How to clone the repo:
```
$ git clone https://github.com/jen0828/AJAX.git
$ cd AJAX
```
How to run Random Joke API app:
```
$ cd RandomJokeAPi
$ open index.html
```
How to run TV Show API app:
```
$ cd TVShowApi
$ open index.html
```
